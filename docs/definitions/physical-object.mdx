## Summary

- **Hex Code:** `43`
- **Standard Alignment:** [`schema:Product`](https://schema.org/Product) + [`schema:Thing`](https://schema.org/Thing) (Close fit)

<Callout type="warning" title="Not">
  Not a location (where). Not a digital artifact (use CreativeWork).
</Callout>

## Fide ID Rules

### As Entity Type

- MUST use [Reference Type](/fide-id/specification/index#reference-type) [`NetworkResource`](/vocabulary/definitions/network-resource).
