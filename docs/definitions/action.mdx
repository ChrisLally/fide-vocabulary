## Summary

- **Hex Code:** `42`
- **Standard Alignment:** [`schema:Action`](https://schema.org/Action) + [`prov:Activity`](https://www.w3.org/ns/prov#Activity) (Close fit)

<Callout type="warning" title="Not">
  Not a broad temporal happening (Event); must be driven by an Agent.
</Callout>

## Fide ID Rules

### As Entity Type

- MUST use [Reference Type](/fide-id/specification/index#reference-type) [`NetworkResource`](/vocabulary/definitions/network-resource).
