## Definition

A formal concept, word, topic, category, or abstract idea defined in a taxonomy.

- **Boundaries:** Not a plain raw value.
- **Hex Code:** `31`
- **Standard Alignment:** [`schema:DefinedTerm`](https://schema.org/DefinedTerm) (Close fit)


## Fide ID Rules

These rules define how Concept can appear inside a [Fide ID](/fide-id).

### As Entity Type

- MUST use [Reference Type](/fide-id/specification/index#reference-type) [`NetworkResource`](/vocabulary/definitions/network-resource).





## Examples

| Status | Entity Type | Reference Type | Fide ID Prefix |

| Valid | `Concept` | `NetworkResource` | `did:fide:0x3120` |
| Invalid | `Concept` | `Statement` | `did:fide:0x3100` |
