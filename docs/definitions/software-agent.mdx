## Definition

A running process that acts as an independent actor: bot, AI agent, smart contract. Can hold keys and sign.

- **Boundaries:** Not a person or organization. Not static passive code (use CreativeWork for repos, images, logic).
- **Hex Code:** `12`
- **Standard Alignment:** [`schema:SoftwareApplication`](https://schema.org/SoftwareApplication) + [`prov:SoftwareAgent`](https://www.w3.org/ns/prov#SoftwareAgent) (Close fit)


## Fide ID Rules

These rules define how SoftwareAgent can appear inside a [Fide ID](/fide-id).

### As Entity Type

- MUST use [Reference Type](/fide-id/specification/syntax#reference-type) [`NetworkResource`](/vocabulary/definitions/network-resource).
