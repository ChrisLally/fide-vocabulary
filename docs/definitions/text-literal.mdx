## Summary

- **Hex Code:** `a0`
- **Standard Alignment:** [`rdf:Literal`](https://www.w3.org/1999/02/22-rdf-syntax-ns#Literal) + [`xsd:string`](https://www.w3.org/2001/XMLSchema#string) (Exact fit)

<Callout type="warning" title="Not">
  Not a Concept.
</Callout>

## Fide ID Rules

### As Entity Type

- MUST use [Reference Type](/fide-id/specification/index#reference-type) [`TextLiteral`](/vocabulary/definitions/text-literal) or [`NetworkResource`](/vocabulary/definitions/network-resource).




### As Reference Type

- [Reference](/fide-id/specification/index#reference) MUST use the exact text value.
