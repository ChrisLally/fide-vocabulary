## Summary

- **Hex Code:** `11`
- **Standard Alignment:** [`org:Organization`](https://www.w3.org/ns/org#Organization) (Exact fit)

<Callout type="warning" title="Not">
  Not a single person or software. Not a query-defined group (e.g. alumni by year); create only when the group itself needs to act.
</Callout>

## Fide ID Rules

### As Entity Type

- MUST use [Reference Type](/fide-id/specification/index#reference-type) [`NetworkResource`](/vocabulary/definitions/network-resource).
