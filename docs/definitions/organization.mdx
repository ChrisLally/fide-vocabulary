## Definition

A structured collective of people or agents that acts as a unit.

- **Boundaries:** Not a single person or software. Not a query-defined group (e.g. alumni by year); create only when the group itself needs to act.
- **Hex Code:** `11`
- **Standard Alignment:** [`org:Organization`](https://www.w3.org/ns/org#Organization) (Exact fit)


## Fide ID Rules

These rules define how Organization can appear inside a [Fide ID](/fide-id).

### As Entity Type

- MUST use [Reference Type](/fide-id/specification/index#reference-type) [`NetworkResource`](/vocabulary/definitions/network-resource).
